## 1. 什么是技能编辑器

角色工作室的技能编辑器允许你**不写任何代码**，通过点击和填写数字，创建可以在游戏中实际使用的技能。

这些技能可以：
- 对敌人造成伤害（多种伤害类型）
- 治疗己方殖民者
- 施加强化 / 弱化效果（Hediff）
- 召唤生物
- 让角色瞬间位移
- 控制敌人（使其晕眩）
- 改变地形（挖墙、填坑）
- 播放粒子特效（尘雾、火焰、爆炸等）

你制作的技能可以：
- 直接绑定给地图上的任何角色立即使用
- 导出为 RimWorld 模组文件永久保存
- 与皮肤一起保存，作为角色套装的一部分

## 2. 打开技能编辑器

**方法一：通过皮肤编辑器**

1. 进入游戏，点击底部的「角色工作室」按钮（画笔图标）
2. 在皮肤编辑器顶部菜单栏找到「技能编辑器」按钮
3. 点击打开

**方法二：通过角色右键菜单**

1. 选中地图上的殖民者
2. 右键点击底部的「外观」Gizmo 按钮
3. 选择「打开皮肤编辑器」，进入后再打开技能编辑器

## 3. 界面总览

技能编辑器分为三个区域：

```
+------------------+-------------------------+------------------------------------+
|                  |                         | [ 效果 (N) ][ VFX (N) ][ RC (N) ] |
|  左侧：技能列表   |  中间：基础属性 + 载体    +------------------------------------+
|                  |                         |                                    |
| * 显示所有技能   | * 名称、图标、冷却         | 效果标签：效果列表                  |
| * 新建/复制/删除  | * 技能类型（怎么打出去）   | VFX标签：视觉特效列表               |
| * 搜索技能       | * 射程、范围              | RC标签：运行时组件列表              |
|                  | * 校验按钮               |                                    |
|  顶部：绑定角色  |                          |                                    |
+------------------+-------------------------+------------------------------------+
```

### 右侧面板三标签说明

右侧面板有三个标签页，括号内显示当前数量：

| 标签 | 内容 | 用途 |
|------|------|------|
| **效果 (N)** | 效果列表 | 配置技能命中后的实际效果（伤害、治疗等） |
| **VFX (N)** | 视觉特效列表 | 配置粒子特效（尘雾、火焰、闪电等） |
| **运行时组件 (N)** | 高级组件列表 | 配置连段、短跳、爆炸叠加等复杂机制 |

### 状态说明

| 状态 | 含义 |
|------|------|
| 校验通过 | 技能配置正确，可以使用 |
| 有警告 | 技能有警告（可以用，但建议检查） |
| 有错误 | 技能配置有错误，无法正常使用 |

## 4. 创建第一个技能：快速上手

我们来做一个最简单的技能：**近战重击** —— 对相邻的敌人造成 30 点钝击伤害，并播放尘雾特效。

### 第一步：新建技能

点击左上角的「新建」按钮，列表中会出现一个叫「新技能」的条目，同时右边面板变为可编辑状态。

### 第二步：填写基本信息

在中间的「基础设置」区域：

- **DefName**：填入 `MyHeavyStrike`
  （只能用英文字母、数字和下划线，这是技能的内部 ID，不影响显示）
- **名称**：填入 `重击`
- **冷却（Tick）**：填入 `300`（300 Tick 约 5 秒）
- **预热（Tick）**：填入 `60`（1 秒预热）
- **次数**：填入 `1`

### 第三步：选择载体类型

在「载体设置」区域，点击「类型」下拉菜单，选择 **Target（指定目标）**。

此时会出现射程输入框，填入 `2`（近战距离）。

### 第四步：添加伤害效果

切换到右侧「效果」标签，点击「添加效果」按钮，选择 **Damage（伤害）**。

在弹出的效果卡片中：
- **数值**：填入 `30`
- **几率**：填入 `1`（100% 触发）
- **伤害类型**：点击按钮，从列表中选择 **Blunt（钝击）**

### 第五步：添加视觉特效（可选）

切换到右侧「VFX」标签，点击「添加特效」按钮，选择 **DustPuff（尘雾）**。

保持默认参数即可（目标：Target，延迟：0 Tick，规模：1.0）。

### 第六步：校验

点击中间面板底部的「校验」按钮，如果显示「校验通过」，说明技能配置正确。

恭喜！你的第一个技能做好了。

## 5. 基础属性详解

| 字段 | 说明 | 填写建议 |
|------|------|----------|
| DefName | 技能的内部 ID，全局唯一 | 用英文，加上你的前缀，如 `MyMod_FireBolt` |
| 名称 | 游戏内显示的名称 | 任意中文 / 英文 |
| 图标路径 | 技能按钮的图标 | 留空使用默认图标；填入游戏内图标路径如 `UI/Abilities/Shoot` |
| 冷却（Tick） | 使用后多久才能再次使用 | 300=5秒，600=10秒，1800=30秒 |
| 预热（Tick） | 释放前的准备时间 | 0=即时，60=1秒，120=2秒 |
| 次数 | 可以使用几次 | 通常填 `1` |

> **Tick 换算**：RimWorld 每秒约 60 Tick。300 Tick = 5 秒，1800 Tick = 30 秒，3600 Tick = 1 分钟。

## 6. 载体类型（技能怎么打出去）

这是最关键的选择，决定了玩家怎么使用这个技能：

### Target（指定目标）

- **用途**：点击一个目标（敌人、队友或地块）来释放
- **参数**：射程（多远能打到）
- **典型用途**：近战攻击、单体治疗、单体控制

### SelfCast（自身施放）

- **用途**：点击技能按钮后立即对自己生效，无需选择目标
- **参数**：无额外参数
- **典型用途**：自身 Buff、防御姿态

### AOE（范围效果）

- **用途**：以释放者为圆心，对周围所有格子生效
- **参数**：范围半径
- **典型用途**：范围攻击、群体治疗、地形改造

### Projectile（抛射物）

- **用途**：发射一个抛射物飞向目标
- **参数**：射程 + 抛射物 Def（从列表中选）
- **典型用途**：远程攻击、投掷炸弹

## 7. 效果列表（打中之后发生什么）

在右侧「效果」标签页中管理技能效果。一个技能可以有多个效果，命中时全部依次触发。

### 添加效果

点击「添加效果」按钮，从下拉菜单选择效果类型。

### 效果卡片通用字段

| 字段 | 说明 |
|------|------|
| **数值** | 效果强度（伤害量 / 治疗量等） |
| **几率** | 0.0~1.0，触发概率（1=100%，0.5=50%） |
| 上移 / 下移 | 调整效果执行顺序 |
| **X** | 删除该效果 |

| 参数 | 说明 | 示例 |
|------|------|------|
| 数值 | 治疗点数 | 20 |
| 几率 | 触发概率 | 1 |

## 9. 视觉特效（VFX）系统

在右侧「VFX」标签页中配置技能的粒子特效。视觉特效纯粹是表现层，不影响实际伤害和效果。

### 添加特效

点击「添加特效」按钮，从菜单选择特效类型。

### VFX 卡片参数

| 参数 | 说明 | 可选值 |
|------|------|--------|
| 特效类型 | 播放什么粒子 | 见下表 |
| 作用目标 | 在哪里播放 | Caster / Target / Both |
| 延迟（Tick） | 命中后延迟多少 Tick 再播放 | 0=立即，60=1秒后 |
| 规模 | 特效大小倍率 | 1.0=标准大小 |

### 特效类型说明

| 类型 | 效果描述 | 适用场景 |
|------|----------|----------|
| DustPuff | 尘雾飞溅 | 近战打击、落地冲击 |
| MicroSparks | 细小火星 | 电击、金属碰撞 |
| LightningGlow | 闪电光芒 | 雷电技能、能量爆发 |
| FireGlow | 火焰光辉 | 火焰技能、灼热攻击 |
| Smoke | 烟雾 | 爆炸余烟、隐身技能 |
| ExplosionEffect | 爆炸特效 | 范围爆炸、大招释放 |

### 延迟特效示例

想要先打出伤害，然后 0.5 秒后播放爆炸特效：

1. 添加一个 ExplosionEffect 特效
2. 将延迟（Tick）设为 30（约 0.5 秒）
3. 将作用目标设为 Target

## 10. 运行时组件详解

在右侧「运行时组件（RC）」标签页中配置高级技能机制。

> 运行时组件是高级功能，普通技能不需要配置。

### Q 连段窗口（QComboWindow）

用于 Q 键循环攻击形态。

| 参数 | 说明 | 建议值 |
|------|------|--------|
| 连段窗口（Tick） | 两次 Q 之间允许的最大间隔 | 12（约 0.2 秒） |
| 启用 | 是否激活此组件 | 勾选 |

### E 短跳（EShortJump）

E 键触发短距离闪现位移。

| 参数 | 说明 | 建议值 |
|------|------|--------|
| 冷却（Tick） | 每次短跳的冷却时间 | 120（2 秒） |
| 跳跃距离（格） | 每次位移多远 | 6 |
| 搜索半径 | 落点搜索范围 | 3 |
| 跳后触发效果 | 落地后是否触发技能效果 | 勾选 |

### R 叠加爆发（RStackDetonation）

R 键二段技，第一段积累叠层，第二段触发多波范围爆炸。

| 参数 | 说明 | 建议值 |
|------|------|--------|
| 所需叠层 | 积累到多少层才能引爆 | 7 |
| 延迟（Tick） | 第一段到第二段的延迟 | 180（3 秒） |
| 第一波 半径 / 伤害 | 最内圈爆炸 | 3 / 80 |
| 第二波 半径 / 伤害 | 中圈爆炸 | 6 / 140 |
| 第三波 半径 / 伤害 | 最外圈爆炸 | 9 / 220 |
| 伤害类型 | 三波爆炸使用的伤害 Def | Bomb |

## 11. 复杂技能示例

### 示例 A：群体治疗技能

目标：以自身为中心，5 格内所有队友恢复 25 点生命值，播放柔和光芒特效。

1. 新建技能，DefName = MyAOEHeal，名称 = 圣光治愈
2. 冷却 = 600（10 秒），预热 = 120（2 秒）
3. 载体类型 = AOE，范围 = 5
4. 效果标签 -> 添加 Heal，数值 = 25，几率 = 1
5. VFX 标签 -> 添加 LightningGlow，目标 = Caster，延迟 = 0，规模 = 1.5
6. 点击校验，通过后绑定给治疗者

### 示例 B：闪电打击

1. 新建技能，DefName = MyLightningStrike，名称 = 闪电打击
2. 冷却 = 180，预热 = 30，载体 = Target，射程 = 12
3. 效果标签 -> 添加 Damage，数值 = 40，伤害类型 = Flame
4. VFX 标签：添加 LightningGlow（延迟=0），再添加 MicroSparks（延迟=10）
5. 校验通过

### 示例 C：爆炸冲锋（带短跳 + VFX）

1. 新建技能，DefName = MyDashBlast，名称 = 爆炸冲锋，冷却 = 240，载体 = Target，射程 = 8
2. 效果标签 -> 添加 Damage，数值 = 60，伤害类型 = Bomb
3. VFX 标签 -> 添加 ExplosionEffect，目标 = Target，延迟 = 5，规模 = 1.2
4. RC 标签 -> 添加 EShortJump：冷却 = 240，跳跃距离 = 7，勾选「跳后触发效果」
5. 校验通过，绑定给战士

## 12. 把技能给到角色身上

### 方法一：编辑器内直接绑定

1. 在技能编辑器左上角找到「绑定角色」区域
2. 点击角色选择按钮，从地图上选择目标殖民者
3. 点击「授予技能」—— 所有技能立即生效

### 方法二：通过热键槽绑定

1. 在绑定区域配置 Q / W / E / R 各槽位对应的技能 DefName
2. 点击「授予技能」
3. 游戏中按对应按键触发

注意：未绑定热键的技能也可以从 Gizmo 面板手动点击使用。

## 13. 技能 Gizmo（游戏内按钮）使用说明

绑定技能后，选中角色会在底部出现技能快捷按钮（Gizmo）。

| 按钮状态 | 含义 |
|----------|------|
| 正常 | 技能就绪，可以使用 |
| 灰色 | 冷却中，数字显示剩余时间 |
| 黄色轮廓 | 正在预热 |

点击 Gizmo 按钮或按对应热键（Q/W/E/R）即可使用技能。

## 14. 导出为模组（让技能永久保存）

技能在编辑器中只是临时数据，关闭游戏后会丢失。要永久保存，需要导出为模组：

1. 在皮肤编辑器中点击「导出模组」按钮
2. 填写模组名称和作者
3. 选择导出路径（建议导出到 RimWorld Mods 文件夹）
4. 点击「导出」

导出后的模组包含所有皮肤配置和技能定义，可以直接在模组列表中启用。

## 15. 常见问题 FAQ

**Q：技能校验失败怎么办？**

点击「校验」后，错误信息会显示在按钮下方。常见原因：
- DefName 包含中文或特殊字符
- 伤害类型未选择（显示「无」）
- 效果数值为 0 或负数

**Q：技能绑定后游戏内看不到？**

确认：已点击「授予技能」按钮、角色已正确绑定、技能校验通过。

**Q：VFX 特效没有播放？**

- 确认技能已成功命中目标
- 检查延迟 Tick 是否过大
- 确认作用目标设置正确（Caster/Target/Both）

**Q：Tick 和秒怎么换算？**

RimWorld 正常速度下：1 秒 = 60 Tick。300 Tick = 5 秒，1800 Tick = 30 秒。

**Q：DefName 可以重复吗？**

不可以。同一皮肤内的所有技能 DefName 必须唯一。建议用有意义的英文前缀区分，如 `MyChar_Q_Slash`。

**Q：运行时组件能和效果一起用吗？**

可以。效果列表和运行时组件是独立的两套系统，可以同时配置。
