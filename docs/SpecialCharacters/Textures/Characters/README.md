# Textures Folder - 角色贴图目录

此目录用于存放角色贴图资源。

## 目录结构建议

```
Textures/
└── Characters/
    └── ExampleCharacter/
        ├── Portrait.png       # 角色立绘 (建议 256x256 或 512x512)
        ├── Body.png           # 身体贴图
        ├── Head.png           # 头部贴图
        └── Icon.png           # 角色图标 (建议 128x128)
```

## 贴图规范

### 格式要求
- **文件格式**: PNG（支持透明背景）
- **色彩模式**: RGB 或 RGBA

### 尺寸建议
| 类型 | 推荐尺寸 | 说明 |
|------|----------|------|
| 角色立绘 | 256x256 或 512x512 | 用于角色选择界面 |
| 身体贴图 | 128x128 | 游戏内显示 |
| 头部贴图 | 64x64 | 头像显示 |
| 图标 | 128x128 | 菜单和列表图标 |

### 设计指南
1. **风格统一** - 保持与 RimWorld 原版画风一致
2. **清晰可辨** - 角色特征明显，易于识别
3. **透明背景** - 使用 alpha 通道实现透明
4. **适当留白** - 边缘留出适当空白，避免贴图被裁切

## 贴图引用

在 Def 文件中引用贴图时，路径相对于 `Textures` 目录：

```xml
<iconPath>Characters/ExampleCharacter/Icon</iconPath>
```

注意：
- 路径不包含 `Textures/` 前缀
- 不包含文件扩展名 `.png`
