## 1. 什么是角色编辑器

角色编辑器（皮肤编辑器）是角色工作室的核心功能，允许你为 RimWorld 角色创建完全自定义的外观皮肤，包括：

- **基础外观替换**：替换角色的身体、头部、头发、胡须贴图
- **自定义图层叠加**：在角色身上添加任意数量的额外图层（装饰、纹身、服装配件等）
- **动态表情系统**：根据角色心情、状态自动切换面部表情贴图
- **技能绑定**：将自定义技能与皮肤绑定，随皮肤一起导出
- **AI 辅助生成**：使用 AI 描述角色，自动生成属性配置

制作完成的皮肤可以导出为 RimWorld 模组文件，与其他玩家分享。

## 2. 打开角色编辑器

**方法一：主按钮**

1. 在 RimWorld 游戏内，点击底部工具栏最右侧的「角色工作室」按钮（画笔图标）
2. 角色编辑器窗口将打开

**方法二：通过角色右键菜单**

1. 在地图上选中一个殖民者
2. 右键点击底部的「外观」Gizmo 按钮
3. 选择「打开皮肤编辑器」
4. 编辑器将打开并自动绑定该角色

## 4. 素材准备指南

这是最重要的章节。所有贴图素材必须满足以下规格才能正常显示。

### 4.1 贴图格式要求

| 项目 | 要求 |
|------|------|
| **文件格式** | PNG（必须，支持透明通道） |
| **色彩模式** | RGBA（RGB + Alpha 透明通道） |
| **分辨率** | 需为 2 的幂次方（64、128、256、512、1024 等），推荐至少 512x512，高清种族建议 1024x1024 |
| **背景** | 透明（Alpha=0），不要有白色或其他颜色背景 |
| **文件编码** | 无 ICC 色彩配置文件 |

> 提示：使用 Aseprite、Photoshop 或 GIMP 制作贴图时，确保导出时选择「PNG-24」或「PNG with Alpha」格式。

### 4.2 贴图尺寸规范

角色工作室对贴图尺寸**没有强制限制**，渲染完全由 Unity/RimWorld 引擎处理。

**核心规则：尺寸必须是 2 的幂次方**

允许的尺寸：`64`、`128`、`256`、`512`、`1024`、`2048`（宽高可以不同但都必须是2的幂）

| 规格 | 适用场景 |
|------|----------|
| 64 / 128 | 原版风格像素画，轻量化 |
| 256 / 512 | **推荐最低配置**，兼顾清晰度与性能 |
| 1024 | 高清种族模组主流规格，细节丰富 |
| 2048 | 超高清，仅在特别需要时使用（内存占用大） |

**各部位参考尺寸（支持高清）**

| 部位 | 原版尺寸 | 推荐高清尺寸 | 说明 |
|------|----------|-------------|------|
| 身体（Body） | 64x128 px | 512x1024 / 1024x2048 | 正面+侧面+背面三方向拼合 |
| 头部（Head） | 64x64 px | 512x512 / 1024x1024 | 单方向或四方向贴图 |
| 头发（Hair） | 64x64 px | 512x512 / 1024x1024 | 与头部保持相同尺寸 |
| 胡须（Beard） | 64x64 px | 512x512 / 1024x1024 | 与头部保持相同尺寸 |
| 自定义图层 | 按需 | 按需（2的幂次方） | 根据图层实际内容决定 |
| 表情贴图 | 64x64 px | 512x512 / 1024x1024 | 与头部贴图尺寸一致 |

> **重要提示**：使用高清贴图时，身体贴图的比例关系需保持正确。
> 原版 Body 是 64x128（宽:高 = 1:2），高清版本应保持相同比例如 512x1024 或 1024x2048。

### 4.3 Graphic_Multi 多方向贴图

如果要让图层支持多方向（南/北/东/西不同画面），需要准备 4 张贴图并命名为：

```
MyTexture_south.png    (正面/南方)
MyTexture_north.png    (背面/北方)
MyTexture_east.png     (右侧/东方)
MyTexture_west.png     (左侧/西方，通常与east相同，系统自动翻转)
```

或者：直接提供一张包含四个方向的拼合贴图（宽度 N，高度 4N，从上到下依次为南/北/东/西）。
例如：原版 64x256 px，高清 512x2048 px。

如果只有一个方向，使用 `Graphic_Single` 图形类即可。

### 4.4 贴图放置位置

**方式一：放在游戏 Textures 文件夹（导出模组用）**

建议结构：
```
Mods/
  你的模组名/
    Textures/
      Characters/
        MyCharacter/
          body.png
          head.png
          hair.png
          expressions/
            neutral.png
            happy.png
```

在编辑器中填写路径时，使用相对路径（从 Textures 文件夹开始，不含扩展名）：
```
Characters/MyCharacter/body
Characters/MyCharacter/head
```

**方式二：使用绝对路径（临时测试用）**

填写贴图的完整磁盘路径，如：
```
C:/Users/你的用户名/Desktop/my_texture.png
```

注意：绝对路径只在本机有效，导出模组前必须改为相对路径。

**方式三：使用游戏内原版贴图路径**

可以直接引用游戏内已有的贴图，例如：
```
Things/Pawn/Humanlike/Bodies/Naked_Human
Things/Pawn/Humanlike/Heads/Average_Narrow
UI/Abilities/Shoot
```

### 4.5 贴图浏览器

编辑器内置贴图浏览器，点击任意贴图路径字段旁的「浏览」按钮即可：
- 浏览游戏内所有已加载贴图
- 支持搜索过滤
- 点击选择后自动填入路径

## 5. 基础外观标签页

「基础外观」标签页用于替换角色原版的四个主要部位贴图。

### 各槽位说明

| 槽位 | 替换内容 | 建议贴图尺寸 |
|------|----------|-------------|
| **Body（身体）** | 角色躯干和四肢 | 64x128 px |
| **Head（头部）** | 角色头部轮廓 | 64x64 px |
| **Hair（头发）** | 发型 | 64x64 px |
| **Beard（胡须）** | 胡须或下巴装饰 | 64x64 px |

### 使用步骤

1. 点击想要修改的槽位（如 Body）
2. 在右侧属性面板中点击「贴图路径」字段
3. 手动输入路径，或点击「浏览」从贴图库选择
4. 预览面板会实时显示效果

### 颜色控制

每个槽位支持两种颜色来源：

| 颜色来源 | 说明 |
|----------|------|
| **Fixed（固定颜色）** | 使用你指定的固定 RGB 颜色 |
| **PawnSkin（角色皮肤色）** | 跟随角色的皮肤颜色自动变化 |
| **PawnHair（角色发色）** | 跟随角色的头发颜色自动变化 |
| **None** | 不染色，直接显示贴图原色 |

### 偏移和缩放调整

- 在预览面板中，选中部位后可以**拖拽**调整位置偏移
- 在右侧属性面板中可以精确输入数值
- 支持南/东/北三个方向分别设置偏移

## 6. 自定义图层标签页

「图层」标签页允许在角色身上叠加任意数量的额外贴图图层。

### 图层的应用场景

- 装甲纹路叠加
- 纹身或疤痕
- 头盔装饰
- 翅膀或尾巴
- 武器持握效果
- 发光特效图层

### 添加图层

1. 点击「图层」标签
2. 点击工具栏中的「+（添加图层）」按钮
3. 新图层出现在列表底部
4. 在右侧属性面板配置该图层

### 图层属性说明

| 属性 | 说明 |
|------|------|
| **图层名称** | 给图层起一个描述性名称，方便管理 |
| **贴图路径** | 图层使用的贴图文件（参见素材要求） |
| **锚点标签** | 附加到角色的哪个部位（见下表） |
| **锚点路径** | 精确路径定位，优先级高于锚点标签（如 Root/Body/Head:0） |
| **偏移 XYZ** | 相对于锚点的位置偏移 |
| **东向偏移** | 角色朝东/西时的额外偏移 |
| **北向偏移** | 角色朝北时的额外偏移 |
| **缩放** | 贴图大小倍率（1.0=原始大小） |
| **旋转** | 旋转角度（度） |
| **水平翻转** | 勾选后东西方向自动翻转 |
| **绘制顺序** | 数值越大越靠前显示（越不容易被遮挡） |
| **颜色来源** | Fixed/PawnSkin/PawnHair/None |
| **着色器** | Cutout（默认）/Transparent/CutoutComplex |
| **动画类型** | 无/摇摆/呼吸/弹跳 |
| **图形类型** | Graphic_Single（单张）/Graphic_Multi（多方向） |

### 常用锚点标签

| 锚点标签 | 说明 |
|----------|------|
| Head | 头部（跟随头部运动） |
| Body | 躯干（跟随身体） |
| Hair | 头发位置（比头部略高） |
| Beard | 胡须位置 |

> 如果不确定锚点，默认使用 Body 或 Head 即可，然后通过偏移值调整位置。

### 图层排序

- 使用工具栏的「上移」「下移」按钮调整图层顺序
- 绘制顺序越大的图层显示在越前面（叠在其他图层上方）
- 推荐：身体装饰用 0~10，头部装饰用 10~20，特效图层用 50 以上

### 动画类型

| 类型 | 效果 |
|------|------|
| 无（None） | 静态，不动 |
| 摇摆（Sway） | 左右摇摆，适合翅膀、尾巴 |
| 呼吸（Breathe） | 上下缩放，适合发光光晕 |
| 弹跳（Bob） | 上下移动，适合悬浮配件 |

## 7. 表情配置标签页

「表情」标签页用于配置角色在不同情绪和状态下的面部贴图，实现动态换脸效果。

### 工作原理

表情系统通过**整张替换头部贴图**来实现表情变化。当角色的心情、状态改变时，系统自动切换到对应表情的贴图。

### 支持的表情状态

**情绪类（随心情值变化）**

| 表情 | 触发条件 |
|------|----------|
| Neutral（平静） | 默认状态 |
| Happy（快乐） | 心情 > 80% |
| Cheerful（愉快） | 心情 > 90% |
| Gloomy（阴郁） | 心情 20%~40% |
| Sad（悲伤） | 心情 < 20% |
| Hopeless（绝望） | 心情 < 10% |

**生理状态类**

| 表情 | 触发条件 |
|------|----------|
| Tired（疲劳） | 休息值 < 15% |
| Pain（痛苦） | 倒地 / 濒死 |
| Sleeping（睡眠） | 在床上睡觉 |

**战斗状态类**

| 表情 | 触发条件 |
|------|----------|
| Angry（愤怒） | 心理崩溃 / 攻击状态 |
| Scared（恐惧） | 逃跑 / 恐慌状态 |
| Shock（震惊） | 受到攻击 |
| WaitCombat（备战） | 举枪警戒 |
| AttackMelee（近战中） | 进行近战攻击 |
| AttackRanged（远程中） | 进行远程攻击 |

**日常活动类**

| 表情 | 触发条件 |
|------|----------|
| Eating（进食） | 吃东西 / 喝酒 |
| Working（工作） | 制作台工作 |
| Hauling（搬运） | 搬运物品 |
| Reading（阅读） | 阅读书籍 |
| SocialRelax（社交） | 社交放松 |
| Goto（移动） | 行走中 |

**特殊类**

| 表情 | 说明 |
|------|------|
| Blink（眨眼） | 随机触发，优先级最高 |
| Dead（死亡） | 角色死亡时 |

### 配置步骤

1. 点击「表情」标签
2. 点击「启用表情系统」开关
3. 找到要配置的表情行（如 Happy）
4. 点击该行的「设置贴图」按钮
5. 输入或浏览选择对应贴图路径

### 帧动画表情

每个表情支持多帧动画：

1. 在某个表情的配置区域点击「添加帧」
2. 为每一帧设置：贴图路径 + 持续时间（Tick）
3. 系统会自动循环播放

**帧动画示例**（眨眼动画）：
- 帧1：正常眼睛贴图，持续 300 Tick（5 秒）
- 帧2：半闭眼贴图，持续 3 Tick
- 帧3：闭眼贴图，持续 3 Tick
- 帧4：半闭眼贴图，持续 3 Tick

### 所需素材

每个表情需要准备一张头部贴图，规格与基础头部贴图相同：
- 尺寸：64x64 px（推荐）
- 格式：PNG，带透明通道
- 命名建议：`head_neutral.png`、`head_happy.png` 等

## 9. 预览面板操作

预览面板位于编辑器中央，实时展示角色当前外观。

### 交互操作

| 操作 | 效果 |
|------|------|
| 左键拖拽 | 旋转角色朝向（南/北/东/西） |

| 点击预览中的部位 | 自动选中对应图层 |

### 预览模式

- **表情预览**：点击「预览表情」按钮，可预览特定表情状态

### 参考鬼影图

如果上传了参考图片，预览面板会显示半透明的参考底图，方便对齐自定义贴图。

## 10. 皮肤管理（新建/保存/导入）

### 新建皮肤

点击顶部工具栏的「新建」按钮，创建一个空白皮肤。

### 保存皮肤

皮肤数据保存在游戏存档中。

### 从 XML 文件导入皮肤

1. 点击「导入」按钮
2. 在文件浏览器中选择 `.xml` 格式的皮肤文件
3. 导入后皮肤立即可用

### 从模组导入

如果安装了包含皮肤定义的模组，可以在「导入」菜单中从模组列表中选择。

## 11. 导出为模组

将皮肤导出为 RimWorld 模组文件，可以与他人分享或长期保存。

### 导出前检查

1. 确认所有贴图路径使用**相对路径**（不要用本地绝对路径）
2. 确认贴图文件已放置在正确位置
3. 皮肤名称和 DefName 不含中文或特殊字符

### 导出步骤

1. 点击顶部工具栏「导出模组」按钮
2. 填写模组信息：
   - **模组名称**：显示在模组列表中的名字
   - **作者**：你的名字
   - **模组 ID**：唯一标识符（如 `YourName.MyCharacter`）
3. 选择导出路径（推荐选择 RimWorld 的 Mods 文件夹）
4. 点击「导出」

### 导出内容

导出的模组包含：
- `About/About.xml`：模组描述文件
- `Defs/`：皮肤定义 XML
- 技能定义 XML（如果有绑定技能）
- 说明文件

> 注意：贴图文件不会自动复制，需要手动将贴图放入模组的 `Textures/` 文件夹。

## 12. 绑定皮肤给角色

### 方法一：在编辑器中绑定

1. 在皮肤编辑器底部状态栏找到「绑定角色」区域
2. 点击「选择角色」，从地图上点击目标殖民者
3. 点击「应用皮肤」—— 皮肤立即生效

### 方法二：通过角色右键菜单

1. 选中殖民者
2. 右键点击「外观」Gizmo 按钮
3. 选择「切换皮肤」
4. 从列表中选择目标皮肤

## 13. 完整制作流程示例

以「狐娘角色」为例，从零开始制作完整皮肤。

### 需要准备的素材

1. `foxgirl_body.png`（512x1024 px，或原版 64x128 px）—— 狐娘身体（宽高比 1:2）
2. `foxgirl_head.png`（512x512 px，或原版 64x64 px）—— 狐娘头部
3. `foxgirl_hair.png`（512x512 px，或原版 64x64 px）—— 狐狸耳朵+头发
4. `foxgirl_tail.png`（256x512 px，或原版 32x64 px）—— 尾巴图层（宽高比 1:2）
5. `face_neutral.png`（与头部相同尺寸）—— 平静表情
6. `face_happy.png`（与头部相同尺寸）—— 开心表情
7. `face_sad.png`（与头部相同尺寸）—— 难过表情

> 所有尺寸必须是 2 的幂次方。高清版本推荐至少 512px；如果种族模组本身是 1024px，表情贴图也应使用 1024x1024 px。

### 第一步：新建皮肤

打开角色编辑器，点击「新建」，给皮肤起名 `FoxGirl_Kitsune`。

### 第二步：配置基础外观

切换到「基础外观」标签：
- Body 槽 -> 贴图路径：`Characters/FoxGirl/foxgirl_body`
- Head 槽 -> 贴图路径：`Characters/FoxGirl/foxgirl_head`
- Hair 槽 -> 贴图路径：`Characters/FoxGirl/foxgirl_hair`

在预览面板确认显示正常。

### 第三步：添加尾巴图层

切换到「图层」标签，点击「+」添加图层：
- 图层名：`FoxTail`
- 贴图路径：`Characters/FoxGirl/foxgirl_tail`
- 锚点标签：`Body`
- 偏移 Y：`-0.3`（尾巴在身体下方）
- 偏移 X：`0.2`（稍微偏右）
- 绘制顺序：`-1`（在身体后面）
- 动画类型：`摇摆`

### 第四步：配置表情

切换到「表情」标签，启用表情系统：
- Neutral -> `Characters/FoxGirl/Expressions/face_neutral`
- Happy -> `Characters/FoxGirl/Expressions/face_happy`
- Sad -> `Characters/FoxGirl/Expressions/face_sad`

### 第五步：整理素材文件

将所有贴图放入导出模组目录：
```
Mods/FoxGirl/
  Textures/
    Characters/
      FoxGirl/
        foxgirl_body.png
        foxgirl_head.png
        foxgirl_hair.png
        foxgirl_tail.png
        Expressions/
          face_neutral.png
          face_happy.png
          face_sad.png
```

### 第六步：导出模组

点击「导出模组」，填写：
- 模组名：`Fox Girl - Kitsune`
- 作者：你的名字
- 模组 ID：`YourName.FoxGirl`

导出后在 Mods 文件夹确认文件齐全。

## 14. 常见问题 FAQ

**Q：贴图路径正确但预览不显示？**

- 确认贴图已放入游戏或模组的 Textures 文件夹
- 路径不含扩展名（不要写 .png）
- 路径分隔符使用斜杠 /
- 游戏需要重新加载才能识别新贴图（或使用绝对路径测试）

**Q：贴图显示有白色边缘？**

贴图导出时选择了白色背景而非透明背景。
重新导出，确保使用 PNG-24 格式，背景透明（Alpha=0）。

**Q：图层位置显示不对？**

- 检查锚点标签是否正确
- 在预览中拖拽调整偏移值
- 注意南/北/东方向的偏移可以分别设置

**Q：表情不切换？**

- 确认已勾选「启用表情系统」
- 确认皮肤已应用给角色
- 检查贴图路径是否正确

**Q：导出的模组贴图显示为粉红色方块？**

贴图文件未放入模组 Textures 文件夹，或路径与皮肤定义中不匹配。
检查 Textures 文件夹内的目录结构与路径字段中填写的路径是否一致。

**Q：图层在北向不显示？**

部分图层类型（Graphic_Single）不区分方向，使用同一张图。
如果需要北向显示不同内容，改用 Graphic_Multi 并准备四张方向贴图。

**Q：能不能直接用模组里的贴图？**

可以。只要那个模组已安装并加载，就能使用其 Textures 文件夹内的任何贴图路径。

## 15. 素材快速参考

### 贴图规格速查

> 尺寸必须是 2 的幂次方（64、128、256、512、1024 等）。角色工作室本身无尺寸限制，推荐至少 512 规格。

| 用途 | 原版尺寸 | 推荐高清尺寸 | 格式 | 备注 |
|------|----------|-------------|------|------|
| 身体 | 64x128 px | 512x1024 / 1024x2048 | PNG（透明） | 三方向拼合（南/北/东），宽高比 1:2 |
| 头部 | 64x64 px | 512x512 / 1024x1024 | PNG（透明） | 单张或四方向 |
| 头发 | 64x64 px | 512x512 / 1024x1024 | PNG（透明） | 与头部相同 |
| 胡须 | 64x64 px | 512x512 / 1024x1024 | PNG（透明） | 与头部相同 |
| 表情 | 64x64 px | 512x512 / 1024x1024 | PNG（透明） | 替换整个头部，与头部贴图等尺寸 |
| 自定义图层 | 按需 | 按需（2的幂次方） | PNG（透明） | 根据实际内容决定 |

### 路径格式速查

| 类型 | 格式示例 |
|------|----------|
| 模组相对路径 | `Characters/MyChar/body` |
| 游戏内原版路径 | `Things/Pawn/Humanlike/Bodies/Naked_Human` |
| 本地绝对路径（测试） | `C:/Users/你/Desktop/texture.png` |

### 锚点标签速查

| 锚点 | 对应部位 |
|------|----------|
| Body | 躯干 |
| Head | 头部 |
| Hair | 发型位置 |
| Beard | 胡须位置 |

### 颜色来源速查

| 来源 | 说明 |
|------|------|
| Fixed | 使用指定的固定颜色染色 |
| PawnSkin | 跟随角色皮肤色 |
| PawnHair | 跟随角色发色 |
| None | 不染色，显示原色 |

### 着色器速查

| 着色器 | 效果 |
|--------|------|
| Cutout | 边缘硬切，推荐用于大多数贴图 |
| Transparent | 半透明支持，适合发光/半透明效果 |
| CutoutComplex | 支持两种颜色染色（colorTwo） |
